# Google AI Studio Import Notes

هذه الحزمة مجهزة لتُرفع أو تُسحب يدويًا إلى Google AI Studio / Build أو تُفتح في Android Studio.

## الوضع الحالي

- نوع المشروع: Native Android.
- اللغة: Kotlin.
- الواجهة: Jetpack Compose.
- التخزين المحلي: Room Database.
- الحالة: Prototype تشغيلي محلي لسوق أدوية/صيدليات باسم المجمع.

## طريقة الرفع إلى Google AI Studio

1. افتح: https://aistudio.google.com
2. ادخل إلى قسم Build / Apps.
3. اختر إنشاء/استيراد مشروع أو ارفع ملفات المشروع حسب الواجهة المتاحة لديك.
4. اسحب ملف ZIP كاملًا أو ارفعه يدويًا.
5. عند طلب شرح المشروع، انسخ محتوى الملف:
   `AI_STUDIO_PROMPT.md`
6. اطلب من AI Studio: افتح المشروع كما هو، لا تعيد بناءه من الصفر، ولا تغير الـ package أو بنية Room إلا عند الضرورة.

## تحذير مهم

لا تضع مفاتيح API داخل الكود أو داخل GitHub. استخدم `.env` محليًا فقط بناءً على `.env.example`.

## ما يجب أن يفهمه AI Studio

هذا المشروع ليس صفحة واحدة أو تجربة وهمية. فيه دورة طلب محلية، عروض صيدليات، ضمان مالي محلي، نزاعات، وصفات، إشعارات داخلية، واعتماد صيدليات.

المطلوب من AI Studio لاحقًا يجب أن يكون إصلاحًا تدريجيًا، وليس إعادة توليد تطبيق جديد من الصفر.
